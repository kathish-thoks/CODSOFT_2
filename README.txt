# Tic-Tac-Toe AI

A **simple Tic-Tac-Toe game** with **unbeatable AI** using the **Minimax algorithm**.  
This project is written in **Python**, designed for beginners and students to understand basic game AI.

---

## Features

- Play against the computer (AI)
- AI is unbeatable using the Minimax algorithm
- Supports multiple rounds
- Tracks scores: Human vs AI and Draws
- Friendly, beginner-style code with clear comments
- Runs in terminal/console (no GUI needed)

---

2. Navigate to the project folder:
